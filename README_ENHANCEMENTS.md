# FT8 CN 功能增强说明

## 新增功能概述

本次更新为FT8 CN应用添加了多项高级功能，以满足不同场景下的使用需求。

## 功能详细说明

### 1. 动态消息切换
- **功能**: 支持在发射中途切换发射消息，类似电脑端软件的实时切换功能
- **使用**: 在高级配置中启用"动态消息切换"选项
- **效果**: 当正在发射时，选择新的呼叫目标会立即停止当前发射并切换到新消息

### 2. 多槽信息解析
- **功能**: 支持解析和处理多槽位的FT8信息
- **使用**: 在高级配置中启用"多槽信息模式"
- **效果**: 能够同时处理多个时间槽的信息，提高信息处理效率

### 3. 远征模式支持

#### 狐狸模式（Fox Mode）
- **功能**: 适用于远征电台，可同时回复多个台站呼号
- **使用**: 在高级配置中启用"狐狸模式"
- **特点**: 
  - 支持队列模式管理多个呼叫台站
  - 自动处理多个同时呼叫的台站
  - 适合远征DX台使用

#### 猎犬模式（Hound Mode）
- **功能**: 自动寻找并呼叫远征台站
- **使用**: 在高级配置中启用"猎犬模式"
- **特点**:
  - 自动识别带有DX、FOX等修饰符的远征台
  - 优先呼叫远征台站
  - 适合追逐DX的台站使用

### 4. 比赛模式
- **功能**: 支持业余无线电比赛，可自定义呼叫修饰符
- **使用**: 
  1. 在高级配置中启用"比赛模式"
  2. 设置自定义修饰符（如：FIELD、DX、001等）
- **效果**: CQ消息会自动添加比赛修饰符

### 5. 队列模式
- **功能**: 支持呼号队列管理，可同时处理多个台站呼叫
- **使用**: 在高级配置中启用"队列模式"
- **特点**:
  - 最大队列大小可配置（1-50）
  - 实时显示队列状态
  - 支持清空队列操作
  - 与狐狸模式配合使用效果更佳

### 6. 界面优化

#### 选择发射后不自动切换
- **功能**: 选择发射后保持在当前界面，不自动跳转到呼叫界面
- **使用**: 在高级配置中启用"选择发射后不自动切换到呼叫界面"
- **效果**: 类似电脑端软件的操作体验

#### 横屏分屏模式
- **功能**: 横屏时自动启用左右分屏显示
- **布局**: 
  - 左侧：解码部分（消息列表、频谱图等）
  - 右侧：接收界面（类似电脑端右侧面板）
- **自动切换**: 横屏时自动启用，竖屏时自动关闭

## 配置方法

### 访问高级配置
1. 打开应用主界面
2. 进入配置界面
3. 找到"高级配置"选项
4. 根据需要启用相应功能

### 配置建议

#### 远征台站配置
```
✓ 狐狸模式
✓ 队列模式
✓ 多槽信息模式
✓ 动态消息切换
队列大小: 15-20
```

#### 追逐DX配置
```
✓ 猎犬模式
✓ 动态消息切换
✓ 不自动切换到呼叫界面
```

#### 比赛配置
```
✓ 比赛模式
✓ 队列模式
✓ 动态消息切换
比赛修饰符: FIELD (或其他比赛标识)
```

## 注意事项

1. **模式互斥**: 狐狸模式和猎犬模式不能同时启用
2. **性能考虑**: 启用多个高级功能可能会增加CPU使用率
3. **兼容性**: 所有功能都向后兼容，不影响现有的基本功能
4. **配置保存**: 所有配置都会自动保存到数据库中

## 技术实现

### 核心改进
- 增强了`FT8TransmitSignal`类，支持动态消息切换
- 扩展了`GeneralVariables`类，增加新的配置项
- 改进了消息解析逻辑，支持多种模式
- 新增了`AdvancedConfigFragment`高级配置界面

### 数据库更新
- 新增配置项自动保存和读取
- 兼容原有配置数据

### UI改进
- 横屏模式的分屏布局
- 实时状态显示
- 队列信息显示

## 版本兼容性

- **最低Android版本**: Android 6.0 (API 23)
- **推荐Android版本**: Android 8.0+ (API 26+)
- **硬件要求**: 与原版相同

---

*这些增强功能让FT8 CN更接近电脑端软件的功能和用户体验，同时充分利用了移动设备的特性。*
